## <a id="3.30">开坑 3.30</a>

  从2022.3.11号回家开始，自己已经在家里读了20天大学了。今天一下子心血来潮想要写点什么，所以就建了个仓库来记录一下。
    
  先说说个人现状吧。**3.11**从学校回家，因为学校通知全员核酸，所以去外面找，但是每个地方都是堆满了人，至少3-4个小时才能检测到，所以就放弃了。晚上通知小区有无症状封闭，有密接的人家贴封条，其余的可以在小区走动。核酸1/2/4/7/14（传统艺能），有志愿者负责快递和外卖的交接，没有出现管理混乱的情况。做核酸的医生们非常辛苦，一个小区一个小区跑，和志愿者交接的效率也很高。篮球场上摆好台子，拉好隔离线，医生过来直接负责检测就可以了（也有可能是小区比较小，检测的人并不多，所以方便管理。
    
  等第七天**3.20**开始因为前几次没有阳性出现，小区宣布解封。但这个时候学校又有阳性出现，即将处于封闭状态，所以要继续在家学习。过几天根据可靠消息来源，浦东某批发地出现大规模爆发，某些医院出状况，一两天后消息基本证实，周围一圈区域全部封闭。在等周末**3.27**整个浦东封闭，封闭前出现大规模抢菜等等情况，小区也重新封闭，好在家里都有提前准备，所以在保证安全的情况下不愁没饭吃。
    
  大概和在学校里的同学们渡过了很不一样的一段时间吧。学校里的条件更加艰苦，希望大家都能好好的。

#### 一些大众心理观察

  因为疫情封闭出现了不同规模的焦虑。小区里还有亲戚群不断有人转发**虚假消息**，全是感情没有一点逻辑。但就是这些毫无逻辑的内容却真的影响了身边的人，由于缺乏科学知识，长一辈的会开始因此焦虑，散发焦虑。还有网上那些新闻下面全是负面，充满攻击性的评论，被封闭在家的，沪吹沪黑，键政党，以至于我自己被这些情绪严重的影响。本来是觉得这些情绪都与我无关，但是浦东封闭之后这些情绪急剧恶化，而且因为不在学校，能面对面交流一些**无关疫情**的人 == 没有。
    
  焦虑的传染性比BA.2还要强，文字传播，想挡也挡不住。因此我决定和这些情绪划清界限，重新整理一下自己的心情来开始从线上到线下这一段时间的学习。我只想能正常上课，正常在学校，正常运动，正常交际。反思了一下，之前太过于期望能正常生活，所以才导致对封闭下的世界严重的烦躁。
    
  但其实这又是很好的学习机会。因为在家，生活上的压力相比学校要减小很多，有更大的个人空间，更好的物质条件，更安静的学习环境。学习CS的路程非常的漫长和孤独，为什么不用好这一段时间好好提升一下自己的学力呢？
    
  等到能回到学校开始正常上课的时候，我希望自己可以变得很不一样。

  一早起来准备肝实验报告，结果一开VSCode发现自动更新了（今天的万恶之源！！），没等看清更新是什么，字就变得糊的不得了。网上一查要打开的时候把gpu关掉，也就是如下操作

结果这一试就是两个小时，windows上只有`doskey`这样的东西满足功能，而且如果跟Linux一样配置bashrc的话，要修改注册表当中的一个叫command processor的变量，可惜垃圾家庭版又没有，那我只能用别的方法了

- **Linux >> Win10 > Win10家庭版**

我想起了被我遗弃的wsl，对吧，毕竟是ubuntu的内核总有这种功能，结果我在bashrc上折腾了一个小时，结果当我打出`source ~/.bashrc`的时候，离谱的事情发生了，电脑直接死机。重启完了我又开wsl，wsl直接就暴毙了。然后我找了wsl的路径复原了bashrc，试图魔法，结果还是不行，原因是wsl存储的bashrc和我从windows找的好像不太一样，可能是windows系统盘自动复原的问题。不过我倒是知道了**Linux的那一大堆config，就是xxxrc，是自带原版备份的**，所以如果以后真的碰到了什么问题，还有复原的机会

- **重装wsl**

这倒是比较快，大家如果想在windows上使用轻量级的Linux Terminal可以去参考Microsoft的官方文档，讲的非常清楚，不过wsl跟原生的linux shell比还是差远了，只能说杯水车薪吧。然后就是一堆操作，最近好像阿里源和清华源不太好用了，中科大源yyds（我们学校什么时候也搞一个，再不行给我们一人开个服务器也好啊，现在唯一提供的就是水杉Vscode那个单核的，只有6gb内存的，而且而且最关键的是没有root权限的阉割版，效率的话还不如自己电脑跑的快）

然后改了bashrc，把上面一行赋值成vscode就解决了，然后我又发现了Windows的一个垃圾地方，你你你一个系统，Program Files为什么要加空格啊，给人家linux都整不会了（bash说：too many arguments）用ls试试看才发现有空格的文件都是拿单引号包起来的，然后就解决了，这里顺便提醒一个，以下这两个有什么区别？

```bash
alias python="yourPath/python.exe $?"
alias python='yourPath/python.exe $?'
```

如果Github显示的颜色的话一眼就知道了，如果没有的话一个是纯文本，一个是外部变量格式化的，建议都用双引号来削除这个影响，而且顺便削除windows空格的影响

wsl在windows的运行里面是直接可以打开的，Win+R然后直接打wsl，也可以开powershell/cmd然后打wsl，或者打bash也可以，wsl出来的是ubuntu的界面，而bash打开的是类似于cmd的界面

我发现在这个**运行**的界面有很多神奇的功能，比如我打chrome就直接开浏览器了，好牛，所以我查了一下，只要把快捷方式或者运行的文件放在一个文件夹，叫一个好听的名字，然后在**电脑->属性->高级系统设置->环境变量**里面找到Path，添加这个文件，放到第一个（这样比较快）就可以了。记得把快捷方式或者exe换一个能快点打出来的名字，比如微信叫wechat，腾讯会议叫classroom等等，这里随便带不带空格，作为一个好习惯最好不要。

- **肝实验报告**（哈希表）

说真的我们学院的算法这门课的实验质量真的不是很高。这次测哈希插入和查找的时间，**固定哈希表的大小**？那这也太没意思了吧，也没有给出具体的比例（负载因子），那我觉得有点测了个寂寞，就让我们写一个静态的哈希算法，不过图画出来还是可以的，跑算法也没有花很多时间

这次听助教的话有好好写注释，不过就算这样我也觉得自己没有特别认真，因为我觉得这个实验并没有让我提高多少，如果强行提高难度，用一些很骚的操作那就有点卷了，最多是对这些个算法有一个直观的图示，知道一下哪个跑的更快，每次实验对我来说都是这个感觉。

我是觉得要真的让这些个实验有点意义，那还是有很多改进的地方的

比如首先得把这些要求搞的具体一点吧？每次都要阅读理解。数据范围？要测几个？我到底要测几个？我的算法应该要怎么写，到底需要包含多少函数，要不要按照《算法导论》的伪代码一模一样的写，还是只要意思对了就行？（如果只是交个oj的话随便糊糊就完了，而且就是一个超级大的完全不想看的东西）有哪些库能用，哪些不能用？比如`sort()`函数，对吧，明显有的地方不该调用，有的时候可以用一下偷懒，写heap肯定不能调stl的优先队列吧？而且说真的，如果真的**只写一个基础的数据结构或者算法**，那写好的，比你写的好的跑的快的多了去了，STL里面有那么多硬件优化过的东西不比我自己的强很多？还有网上写的能看的不能看的，精彩的不精彩的，我直接拿下来不就行了吗？实验报告最后的讨论和对比，没有**实际的情景**对算法的应用，除了复述一下知识点和随便吹吹自己的看法，又有什么意义呢？除了我说我书读的好，我的这些分析又有什么依据呢？大部分的都是主观或者是别人讲的我再讲一遍而已吧。

所以说真的我并没有在课内的实验课上找到算法的意义所在，最多有一个算法快慢的直观感受而已。每次我能做的就是练习一下自己的coding能力，不过要叫oj，有很多限制，也没有锻炼到多少。

当然我们学院也并没有成立多久，在课程设置上有不合理的地方是非常正常的，可以理解。以上这段的最终目的是为了让我自己能学到，而且学院的课程设置能更加合理。我准备在学期末的时候找一些老师交流一下这个问题，我的本心是希望我们数据学院能越来越好，毕竟老师人都很好而且很会听意见。（写到这里有点火气上来了，平静一下）

跳了三天绳之后发现自己的身体状况好了很多，不会出现各种不舒服，像之前心脏啊，肩膀啊，现在完全没有问题了。也有可能是休息的很好，所以进步的非常快。第一天连着100个单摇直接撅倒在地上，第二天连着150没什么问题了，今天开始已经能200个做组了，而且跳的时候也没有感觉很喘很酸痛，只有小腿胀的好难受（坐太久不动了）。今天训练量是150*2 + 200\*6组，最后还尝试了一下双摇，也能连着跳好几十个（到最后手没力气了）。目标是单摇无压力上千，以及双摇能做组，每天都跳的话肯定很快就能实现的。

#### 别的什么

今天突然有一种想法就是，有机会一定要去国外求学或者交流。我又想起了那次坐在别人高中课堂里的感觉，给我印象很深很深。我非常想念那种没有任何拘束，自由奔放而且能让你学到的课堂氛围(atmosphere)。并不是说国内的这种就不好，我觉得对我来说就是没有完全地来开发自己的能力，总觉得自己缺了很多的参与感，但是没人讲的话我也是完全不想交流的。很渴望能和更多一样年纪的人来交流自己学了什么，哪里哪里有不明白的，还有各种有的没的想法，就跟高中的时候的大伙一样。相比于单打独斗，我还是更怀念一大群有很多想法的人一起进步的感觉。

我还想晒着大太阳拼尽全力地奔跑和打球，迎着月光、寂静星空和熙攘的人群独自思索，屋外下着大雨和别人谈心，大雪纷飞之时围着暖气和一堆人吃饭玩耍。这些都是我曾经拥有的，但是他们全都不见了。我只有电脑和手机，音响和耳机，无论什么天气都是一个样子的北房间，作为CS人我却不接受电子模拟出来的一切事务，和现实永远是有差距的，还有一颗无趣的灵魂每天重复着几乎一模一样的事情，十几个小时的屏幕使用时间以及八个小时的睡眠。我所能做的也就是麻痹自己一切都好，以及每天都抱有一点点希望，希望能逃离，能改变。

最近没有看书或者看剧什么的，听歌倒是听得很多，以及长时间的思考也没有，是觉得看一些东西的话自己的情绪会有很大的波动，长时间的思考也是这样，虽然会有非常非常多的好处和新的想法冒出来，但我的潜意识告诉我还不是时候。如果说看点东西是能化解无聊的很好的形式的话，那我第一个反对，我并没有感觉到自己无聊，只是觉得目前不适合。

虽然是被封控者的**清明节**，大家一定要在这三天时间里留一点时间，安静下来，拜一下或者想一下自己的老祖宗，不要忘了自己的根在哪里，无论你还记不记得，有没有见过，如果不愿意的话只是放下手头的所有事情，出去安静一下也是很不错的。
